# 《智慧班级综合管理系统》功能设计文档
## 一、 视觉与交互规范 (UI/UX)
设计风格： 清新简约风格，采用暖色调（如浅米色背景、温暖的琥珀色强调色）。

排版细节：

- 名单展示： 序号与姓名紧密排列（例如 01 张三），视为单一视觉对象，不分离。

- 颜色语义： 积分数字使用高对比度颜色（加分：醒目翠绿；扣分：亮眼深红）。

- 按钮质感： 录入积分的加/减按钮采用柔和色系（如浅绿、淡红），降低操作压迫感。

## 二、 组织架构与学生管理
档案维护： 包含学号、姓名、性别、联系方式、班委职位。支持单条增删改查及 Excel 一键导入。

小组逻辑：

- 支持自由创建/解散小组及人员分配。

- 积分随人走： 调组不影响个人分，小组积分按该组内当前成员的个人总积分计算平均值。

## 三、 核心积分引擎
多维班规： 支持自定义维度（学习、纪律、卫生、体育等）及对应分值。

批量录入流：

- 范围支持： 单人、小组、全班。

- 快捷操作： 系统自动置顶显示 最近使用的 N 条 积分项。

- 备注溯源： 录入时可选填具体原因。

- 存储逻辑： 积分终身累加。溯源明细看板仅保留 30 天记录，超期自动清理，不影响总分统计。

## 四、 课堂互动工具 (核心增强)
智能随机点名：

- 公平模式： 开启后系统自动记录已点人员，排除已点学生，直到本轮（或全班）点完为止。

- 联动功能： 点名结束后可直接点击姓名进入积分录入界面。

沉浸式计时器：

- 多模式： 支持正计时（讨论）与倒计时（限时做题）。

- 全屏支持： 计时期间支持一键全屏，数字大字体居中，适合后排学生观看。

- 预设项： 提供 1/5/10/20 分钟快速启动。

## 五、 统计排名与奖惩
动态排行榜：

- 维度： 支持总分榜、月度榜、维度细分榜。

- 高亮逻辑： 班主任设置“前 N 名”阈值，系统实时高亮标记。

- 并列处理： 分值相同则全部高亮，不占用后续名次额度。

- 数据导出： 支持导出“月度积分排名汇总表” (Excel)，含职位、小组及奖惩标记。

## 六、 安全与运维
身份认证： 预设账号密码登录，不设注册。

维护通道： 密码存储于配置文件，若遗忘由运维人员通过后台直接查阅或重置。
